# Распознавание жестов русского жестового языка (РЖЯ)

Это прототип веб-сервиса, который определяет буквы русского жестового языка по фотографии. Проект сделан в рамках учебного задания, чтобы продемонстрировать полный цикл ML-разработки: от сбора собственного датасета до деплоя веб-приложения.

## Суть метода

Вместо того чтобы обучать тяжелую сверточную нейросеть на пикселях изображения, мы выбрали более лёгкий и эффективный подход:

1. **MediaPipe** находит руку на изображении и строит её "скелет" из 21 ключевой точки (суставы пальцев, запястье).
2. Координаты этих точек нормализуются (чтобы жест определялся независимо от положения руки в кадре).
3. Обученный классификатор (**Random Forest**) принимает эти координаты и определяет букву.

Такой подход позволяет модели работать практически мгновенно даже на слабых процессорах (CPU) и дает высокую точность при сравнительно небольшом датасете.

## Данные и обучение

Датасет был собран самостоятельно с помощью камеры смартфона.

- Всего собрано около **1300 изображений** для 20 букв.
- Реализована предобработка данных: скрипт умеет работать с русскими названиями папок и автоматически вращает изображения, если метаданные камеры телефона перевернули кадр.
- Точность модели на тестовой выборке составляет более **90%**.

## Структура проекта

- `app.py` — код веб-интерфейса на Streamlit.
- `create_dataset.py` — скрипт, который проходит по папкам с фото, извлекает координаты точек руки и сохраняет их в файл.
- `train_model.py` — загружает обработанные данные, обучает модель и сохраняет её веса.
- `data/` — папка с исходными фотографиями (разбиты по папкам). Мы не включили её в репозиторий из-за большого веса.

## Как запустить

Для работы потребуется Python 3.12 (может новее подойдёт, не знаю)

0. **Перед установкой библиотек создайте и активируйте виртуальное окружение:**

   ```bash
   python -m venv .venv
   ./.venv/Scripts/activate
   ```

1. **Установите библиотеки:**

   ```bash
   pip install opencv-python-headless mediapipe scikit-learn streamlit numpy pillow
   ```

   или проще:

   ```bash
   pip install requirements.txt
   ```

2. **Запустите приложение:**
   Если вы просто хотите проверить работу (модель `model.p` уже должна быть в папке):

   ```bash
   streamlit run app.py
   ```

   Сайт откроется в браузере автоматически.

3. **(Опционально) Переобучение модели:**
   Если вы добавили новые фото в папку `data`, выполните команды по очереди:

   ```bash
   python create_dataset.py
   python train_model.py
   ```
